---
description: Otimização agressiva de performance para carregar em menos de 1 segundo
---

Siga estes passos:
1. Identifique gargalos de performance (imagens, JS pesado, render-blocking, etc.)
2. Sugira e aplique:
   - Critical CSS inline
   - Lazy loading + native lazy
   - Otimização de imagens (WebP/AVIF, srcset, sizes)
   - Code splitting e tree shaking
   - Defer/async de scripts não críticos
   - Minificação e compressão (Gzip/Brotli)
   - Preload de recursos críticos
   - Redução de reflows e repaints
3. Forneça o código final já otimizado
4. Liste as métricas esperadas após as otimizações

Sempre priorize performance mobile-first.
